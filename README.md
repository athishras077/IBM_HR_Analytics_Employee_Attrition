# IBM HR Analytics Employee Attrition Analysis

## Project Overview

This project is an end-to-end Data Analytics case study based on the IBM HR Analytics Employee Attrition & Performance dataset.

The objective of this project is to identify the key factors contributing to employee attrition and provide data-driven recommendations that can help HR teams improve employee retention.

The project follows the complete data analytics lifecycle, including business understanding, data cleaning, exploratory data analysis (EDA), SQL-based business analysis, and an interactive Power BI dashboard.

## Key Insights

* The overall employee attrition rate is **16.12%**.
* Employees working overtime show significantly higher attrition.
* Lower-income employees tend to leave more frequently.
* Employees with fewer years at the company experience higher attrition.
* Attrition varies across departments and job roles.
* Job satisfaction and work-life balance are important factors influencing employee retention.

---

## Business Recommendations

* Reduce excessive overtime to minimize employee burnout.
* Strengthen onboarding and engagement programs for new employees.
* Review compensation strategies for lower-income employee groups.
* Improve employee satisfaction through regular feedback and recognition programs.
* Develop targeted retention strategies for departments and job roles with higher attrition.

## How to Run

1. Clone this repository.

```bash
git clone <repository-url>
```

2. Install the required Python libraries.

```bash
pip install pandas numpy matplotlib
```

3. Open the Jupyter Notebook located in the `notebooks` folder.

4. Execute the notebook sequentially to reproduce the analysis.

5. Import the cleaned dataset into MySQL to run the SQL scripts.

6. Open the Power BI (`.pbix`) file to explore the interactive dashboard.
